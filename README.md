# Adult Income Classification using Random Forest

This project aims to predict whether an individual's annual income exceeds $50,000 based on census data. I implemented a **Random Forest Classifier** and optimized its performance through hyperparameter tuning.

## Project Overview
- **Dataset:** [Adult Income Dataset](https://www.kaggle.com/datasets/lodetomasi1995/income-classification)
- **Primary Model:** Random Forest Classifier
- **Optimization:** RandomizedSearchCV
- **Final Accuracy:** 86%

## Tech Stack
- **Language:** Python
- **Environment:** PyCharm / Jupyter Notebook
- **Libraries:** Pandas, Scikit-Learn, Matplotlib, Seaborn

## Model Performance & Tuning

The baseline model started with an accuracy of **85%**. After performing hyperparameter tuning using `RandomizedSearchCV`, the accuracy improved to **86%**.
